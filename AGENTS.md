# AGENTS.md

## プロジェクト概要

このリポジトリでは、Hired or Fired というAIコードレビュー面接ゲームを実装する。

目的は、AIが書いたコードを人間が読み、疑い、要件とのズレを見抜く力を鍛えること。

## 実装方針

- MVP完成を優先する
- ネット対戦、CPU対戦、ログイン、DB、AI採点は実装しない
- 問題データはTypeScriptオブジェクトで管理する
- ゲーム判定とスコア計算は純粋関数にする
- UIコンポーネント内にスコア計算を埋め込まない
- 問題コードは文字列として表示し、実行しない
- `dangerouslySetInnerHTML` は使わない
- LocalStorageはベストスコア、ベストランク、プレイ回数、最終リザルト、音設定に使う

## UI方針

- 世界観は「AIコードレビュー面接ゲーム」
- タイトルは Hired or Fired
- サブタイトルは AI Code Review Challenge
- コード画面はVS Code風
- スコア、ログ、リザルトは少しアーケード感を持たせる
- 正解時はチェック付きグレーアウト、スコアポップ、音、ログ追加を行う
- リザルトにはCSSベースのResult Illustrationを表示する
- デスクトップ優先。モバイルは最低限崩れない程度でよい

## テスト方針

最低限、以下をテストする。

- スコア計算
- Issue範囲判定
- 階層選択の正誤判定
- 全Issue解決判定
- Hiring Rank計算
- Review Rate計算
- LocalStorage helper
- 問題データ整合性
